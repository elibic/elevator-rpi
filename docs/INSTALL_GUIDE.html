🛗 מדריך התקנה — מערכת מעלית RFID

מדריך מלא להתקנה על Raspberry Pi חדש, בפקודה אחת. זמן משוער: 15–20 דקות.

שלושה דברים שכדאי שיהיו מוכנים לפני שמתחילים:
  1. כתובת ה-Firebase (ה-URL של מסד הנתונים).
  2. שם המעלית (ELEVATOR_ID — למשל A או B).
  3. ה-SECRET_KEY (מפתח הכתיבה ל-Firebase).

0 הכנה

  • התקן Raspberry Pi OS על כרטיס ה-SD (עם Raspberry Pi Imager), הדלק את ה-Pi.
  • ודא שה-Pi מחובר לאינטרנט (Wi-Fi או כבל רשת).
  • חבר את קורא ה-RFID ליציאת USB.

1 יצירת טוקן GitHub (פעם אחת)

הריפו פרטי, אז ה-Pi צריך "סיסמה מיוחדת" (טוקן) לקריאה בלבד:

  1. GitHub → תמונת הפרופיל (ימין למעלה) → Settings.
  2. בתפריט הצד גלול למטה → Developer settings.
  3. Personal access tokens → Fine-grained tokens → Generate new token.
  4. שם: elevator-pi · תוקף: לבחירתך (למשל שנה).
  5. Repository access → Only select repositories → בחר elevator-rpi.
  6. Permissions → Repository permissions → Contents → Read-only.
  7. Generate token → העתק ושמור את הטוקן (מתחיל ב-github_pat_...).
⚠️ הטוקן מוצג פעם אחת בלבד. שמור אותו במקום בטוח. אם איבדת — פשוט צור חדש.

2 הורדת הקוד ל-Pi

פתח טרמינל על ה-Pi והקלד (החלף את <TOKEN> בטוקן שלך):

git clone https://<TOKEN>@github.com/elibic/elevator-rpi.git ~/elevator-RFID
cd ~/elevator-RFID

3 הרצת ההתקנה

אפשרות א' — אשף בטרמינל:

sudo ./setup.sh

אפשרות ב' — כלי גרפי בדפדפן (מומלץ למתחילים):

sudo ./setup.sh --web
הפקודה עושה הכל לבד ובסדר הנכון: מושכת קוד עדכני → דרייבר הקורא (CP210x) → הרשאות → תוכנה ותלויות → שני שירותי הרקע → Raspberry Pi Connect → הפעלה.

4 מילוי ההגדרות

האשף (טרמינל או דפדפן) יבקש:

  • FIREBASE_URL, ELEVATOR_ID, SECRET_KEY — מהפרטים שהכנת.
  • מיפוי תגים: קרב כל תג RFID לקורא, ולחץ "סרוק תג"/ענה בטרמינל, ואז הקלד את מספר הקומה. חזור לכל התגים.
  • התראות: כבר לא מוגדרות כאן — הן מנוהלות מרכזית מדשבורד האדמין (🔔 התראות לכל פרויקט).

5 חיבור ל-Raspberry Pi Connect

בסוף ההתקנה יוצג קישור אימות. פתח אותו בדפדפן ואשר עם חשבון ה-Raspberry Pi שלך. מעכשיו אפשר להיכנס ל-Pi מרחוק דרך connect.raspberrypi.com.

6 בדיקה וניהול שוטף

  • בדיקה: python monitor.py --watch — רואים את הקומה והמצב מתעדכנים.
  • על שולחן העבודה נוצר אייקון "הגדרת מעלית RFID" שפותח את הכלי הגרפי (דשבורד: סטטוס, הפעלה/עצירה, לוגים, עדכון מגיט).
עדכון בעתיד: אחרי ששינית קוד ב-GitHub, על ה-Pi פשוט הרץ שוב sudo ./setup.sh (או לחץ "עדכן מגיט" בדשבורד). ההגדרות שלך לא נדרסות.

🛠️ פתרון תקלות

  • הקורא לא מזוהה / אין /dev/ttyUSB0: נתק וחבר מחדש את הקורא, או אתחל את ה-Pi (שירות הדרייבר רץ באתחול).
  • שגיאת הרשאה ב-clone: בדוק שהדבקת את הטוקן נכון ושיש לו גישת Read ל-elevator-rpi.
  • השירות לא רץ: journalctl -u shabbat-detector -f ו-journalctl -u rfid-tracker -f מראים מה קורה.
